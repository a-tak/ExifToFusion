# Claude指示書

このプロジェクトでClaude Code Actionを使用する際の指示です。

## プロジェクト概要
ExifToFusionは、動画や写真のEXIF情報を取得し、各種Fusionタイトルにセットしてタイムラインに追加するPythonスクリプトです。DaVinci Resolve 19 Studioで動作します。

## 言語設定
**重要**: このプロジェクトでは日本語を使用してください。

- 全てのissue、PR、コメントは日本語で作成してください
- コードコメントも日本語で記述してください
- ドキュメントは日本語で作成・更新してください

## 開発ガイドライン

### コーディングスタイル
- Python PEP 8に準拠してください
- 関数名、変数名は英語でも日本語でも構いません
